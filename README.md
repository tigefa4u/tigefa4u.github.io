#### lets begin with test


    git clone https://github.com/tigefa4u/tigefa4u.github.io.git
    cd tigefa4u.github.io
    echo 'install: --no-document' >> ~/.gemrc
    gem install bundle
    bundle install
    touch _config.yml
    jekyll server --watch
    => Now browse to http://localhost:4000
